Mémo
===

Pour installer en version de dev : 
* about:debugging
* "Load temporary add-on"
* Autoriser en mode privée

Soumission pour générer un .xpi :
* Site https://addons.mozilla.org/fr/developers/addons
* Faire un ZIP et le déposer
* Attendre un peu

Installation du xpi :
* Depuis la version desktop : il suffit de le télécharger, il s'intalle
  * Autoriser en mode privée
* Depuis la version mobile Android : cf. https://extensionworkshop.com/documentation/publish/install-self-distributed/#install-addon-from-file-android
  * Download or save the extension file to your device.
  * In Firefox, open Settings, then About Firefox.
  * Tap the Firefox logo five times in quick succession. This unlocks hidden menu items.
  * Open Settings and then Install Extension from File.
  * Browse to and open the saved extension file.
  * When prompted, click Add.
  * The extension appears in the Extension list of installed extensions and is ready to use.
